# Academy Research Hub

Interactive research dashboard analyzing innovative educational models to inform the design of a privatized, individualized educational academy for grades 6–12.

## Vision

Building an academy focused on **applied knowledge** that prepares students for the real world through three pillars:

1. **Academic Core** — Mastery-based, AI-adaptive core subjects
2. **Applied Studios** — AI & Technology, Art & Design, Music (project-based)
3. **Human Development** — Intrapersonal skills, emotional intelligence, life navigation

## Models Analyzed

| Model | Focus | Key Takeaway |
|-------|-------|-------------|
| **Alpha School** | AI-driven adaptive learning | Mastery-based progression + compressed academics |
| **Acton Academy** | Learner-driven education | Socratic method + character-centered development |
| **High Tech High** | Project-based learning | Interdisciplinary projects with real-world audiences |
| **Khan Lab School** | Mastery-based model | Mixed-age learning + student agency |
| **Microschool Model** | Structural format | Small-scale flexibility + personalization |

## Dashboard Features

- **Overview** — Key metrics, radar charts, agency vs. structure comparison
- **Model Comparison** — Grouped bar charts, scalability vs. personalization, cost vs. innovation
- **Pros & Cons** — Expandable cards with full analysis for each model
- **Feature Matrix** — 15-dimension capability comparison across all models
- **Your Blueprint** — Three-pillar curriculum mapping with borrow vs. build-new analysis

## Tech Stack

- HTML / CSS / JavaScript
- Chart.js for data visualization
- No build step required — open `src/index.html` in a browser
